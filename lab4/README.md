# Практическая работа №5: Хеш-таблица (Вариант №5)

## Анализ варианта задания №5

Из документа (стр. 77) вариант №5:
- **Тип ключа**: вещественное число на интервале [10000.0000, +15000.0000]
- **Преобразование**: с точностью 10⁻⁴ (4 знака после запятой) и свёртка к значению k' на интервале [10000, +20000]
- **Метод хеширования**: модульный
- **Метод разрешения коллизий**: квадратичное хеширование

## Запуск приложения

### Требования
- Java 11 или выше
- Maven (опционально, для тестов)

### Компиляция и запуск

```bash
# Компиляция
javac src/*.java

# Запуск
java -cp src Main
```

## Функциональность GUI

1. **Вставка элемента** - добавление пары ключ-значение
2. **Поиск элемента** - поиск значения по ключу
3. **Удаление элемента** - удаление по ключу
4. **Очистка таблицы** - удаление всех элементов
5. **Добавление случайных данных** - генерация 10 случайных элементов
6. **Просмотр структуры** - отображение всех ячеек таблицы
7. **Статистика** - информация о состоянии таблицы

## Теоретические оценки трудоёмкости

| Операция | Оценка |
|----------|--------|
| Поиск (успешный) | ~1/α × ln(1/(1-α)) |
| Поиск (неуспешный) | 1/(1-α) |
| Вставка | ~1/(1-α) |
| Удаление | ~1/(1-α) |

где α = n/m - коэффициент заполнения таблицы
